# Noms des images produits (textiles)

Liste complète des **135 images** à créer et à déposer dans `customizer_frontend/assets/`.

## Convention de nommage

```
{produit}-{couleur}-{vue}.png
```

- **Produits** : `sweatshirt`, `tshirt`, `tshirt-polyester`
- **Vues** : `face`, `dos`, `cote`
- **Couleurs** : voir le tableau ci-dessous

Total : **3 produits × 15 couleurs × 3 vues = 135 images**

## Recommandations pour les images

- Format **PNG**, fond **transparent** de préférence.
- **Même cadrage / même taille** pour les 3 vues d'un même produit (sinon le produit "saute" au changement de vue).
- Garder un cadrage cohérent entre les couleurs (seule la couleur change).
- Carré (1:1) recommandé.

## Notes techniques

- Le code charge automatiquement `{produit}-{couleur}-{vue}.png` au clic sur une couleur et au changement de produit.
- **Repli automatique** : si une image n'existe pas encore, l'image générique actuelle reste affichée (pas d'image cassée). Vous pouvez donc déposer les images progressivement.
- Pour tester : déposez une seule image (ex. `sweatshirt-rouge-face.png`), rechargez la page (Ctrl+F5), cliquez sur Rouge en vue de face.
