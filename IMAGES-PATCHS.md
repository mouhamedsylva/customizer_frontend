# Images des patchs — liste des fichiers à produire

Le configurateur des patchs affiche désormais **une image PNG** par forme et par couleur
(comme les drapeaux). Tant qu'une image de couleur n'existe pas, le configurateur
affiche la version blanche **teintée** automatiquement — donc rien ne casse pendant
que vous produisez les images.

## Où déposer les fichiers
`customizer_frontend/assets/`

## Convention de nommage
`patch-{forme}-{couleur}.png`

- **4 formes** : `rond`, `carre`, `rectangle`, `blason`
- **16 couleurs** (mêmes slugs que les drapeaux)

## Modèle de rendu
Reprenez exactement le style de vos 4 patchs blancs actuels : patch plein, couture
visible sur le pourtour, sur fond **transparent** (ou blanc). La forme doit remplir
le cadre (marges minimales), comme les versions recadrées.
